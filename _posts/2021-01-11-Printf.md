# Printf

#### 기본 형태
```c
int   printf(const char *, ...)
int   printf(고정 매개변수, ...)
```

#### 개념
printf는 사용자가 출력을 좀 더 편하게 할 수 있도록 기능을 제공하는 함수이다. 많이들 써보았지만 그 구조를 잘 아는 사람은 별로 없을 것이다. 따라서 이번 파트에서는 printf의 개념과 특성, 형태 알고리즘에 대해 알아보는 것을 목표로 하고자 한다. 먼저, 들어가기 앞서 가변 인자의 개념에 대해 정리하고 들어가고자 한다.<br/>

#### 가변 인자
가변 인자란 함수에 들어가는 인수의 개수가 변하는 것을 말한다. 즉, 함수에 넣어지는 인자의 수가 정확하게 정해지지 않은 것이다. printf 의 경우 몇 개의 인자를 출력할 것인지 정해져있지 않고 사용자가 원하는 만큼 넣어 사용하게 되는데 이는 가변 인자를 사용하기 때문에 가능한 것이다.<br/>

가변 인자를 정의할 때는 고정 매개변수가 한 개 이상 존재해야 한다. printf의 경우에서 맨 처음에 const char * 형이 온다는 점에서 이를 만족하는 것을 알 수 있다. 고정 매개변수 뒤에 오는 '...'이란 매개변수의 개수가 정해지지 않은 것을 의미한다. 또한, '...' 뒤에 다른 매개변수를 지정할 수 없다는 규칙이 존재한다.<br/>

가변 인자는 stdarg.h 헤더 파일에 정의되어있으며 아래는 가변 인자 처리 매크로를 정리한 것이다.<br/>
- va_list: 가변 인자 목록. 가변 인자의 메모리 주소를 저장하는 포인터입니다.
- va_start: 가변 인자를 가져올 수 있도록 포인터를 설정합니다.
- va_arg: 가변 인자 포인터에서 특정 자료형 크기만큼 값을 가져옵니다.
- va_end: 가변 인자 처리가 끝났을 때 포인터를 NULL로 초기화합니다.
